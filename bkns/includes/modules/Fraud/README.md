# Fraud Module

> Các hàm xử lý liên quan đến phát hiện và phòng chống gian lận (Fraud)

## Chức năng
- Phát hiện giao dịch gian lận
- Xác minh danh tính khách hàng
- Kiểm tra IP / Proxy
- Blacklist management
- Fraud scoring & alerts
