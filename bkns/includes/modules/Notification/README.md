# Notification Module

> Các hàm xử lý liên quan đến thông báo (Notification)

## Chức năng
- Gửi email thông báo
- SMS notification
- Telegram / Webhook alerts
- Template quản lý thông báo
- Lịch sử thông báo

## Cấu trúc files
```
Notification/
├── README.md
├── functions.php          # Các hàm chính
├── hooks.php              # Hook handlers
└── helpers.php            # Utility functions
```
