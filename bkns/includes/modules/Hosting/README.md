# Hosting Module

> Các hàm xử lý liên quan đến dịch vụ hosting

- Tạo / Quản lý tài khoản hosting
- Nâng cấp / Hạ cấp gói hosting
- Gia hạn hosting
- Quản lý cPanel / DirectAdmin
- Backup & Restore
