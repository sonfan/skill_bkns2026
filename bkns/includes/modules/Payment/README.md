# Payment Module

> Các hàm xử lý liên quan đến thanh toán (Payment)

## Chức năng
- Xử lý thanh toán (VNPay, MoMo, Bank Transfer...)
- Tạo / Quản lý hóa đơn (Invoice)
- Hoàn tiền (Refund)
- Payment gateway integration
- Báo cáo doanh thu
